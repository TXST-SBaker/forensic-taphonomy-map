About This Resource & How to Use

This self-contained website aggregates publicly available information on all known active human forensic taphonomy research facilities worldwide (as of April 2026). These outdoor laboratories enable critical studies on postmortem processes across climates, supporting PMI estimation, victim identification, law enforcement training, and disaster response.

Key Data Included
  • Facility name, university/entity, location & precise coordinates
  • Establishment date, current + historical directors
  • Body donation program links & contact info
  • Donor metrics/notes, published research highlights
  • Size, climate/environment, unique features
How to Publish on GitHub (Easy!)
  1. Create new repo on GitHub (e.g. "body-farms-map")
  2. Upload this HTML file as index.html
  3. Go to Settings → Pages → Deploy from "main" branch
  4. Your live site: https://YOURUSERNAME.github.io/body-farms-map/

Single-file = instant hosting. Add CSV to repo for download link.

Important: All facilities are secure, fenced, and closed to the public. Visits by appointment only for qualified researchers/LE. Always respect donor privacy, ethics guidelines, and facility rules. Never attempt unauthorized access.
